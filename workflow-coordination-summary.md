## 🔧 **使用指南**

### 快速开始
1. **访问监控面板**: http://localhost:3001/workflows
2. **查看预定义工作流**: 系统内置4个工作流
3. **执行工作流**: 点击"执行"按钮开始
4. **监控进度**: 实时查看执行状态和结果

### API使用
```bash
# 获取工作流列表
curl http://localhost:3001/api/workflows?action=list

# 执行工作流
curl -X POST http://localhost:3001/api/workflows \
  -H "Content-Type: application/json" \
  -d '{"action":"execute","workflowId":"evening-proactive"}'

# 获取运行中实例
curl http://localhost:3001/api/workflows?action=instances&status=running
```
